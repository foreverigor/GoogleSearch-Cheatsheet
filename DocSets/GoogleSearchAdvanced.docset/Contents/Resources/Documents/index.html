Google Search

Cheat Sheet with Google Search advanced operators.

Special Characters: Summary

term1 term2

Find result with both term1 and term2.

term1 OR term2

term1 | term2

Find result with either term1 or term2 or both (with automatic stemming on).

"term"

Search for term.

Put quotation marks around terms that are stop words — that is, words Google would otherwise ignore — or when you want Google to return only pages that match your search terms exactly.

"phrase"

Find result with the phrase (enclosed in quotes) and * replaced by one or more words.

"term1" "term2"

Disable automatic stemming (Similar Words Match).

i.e., searching for pages that match variants of your search term(s).

"terms1 * terms2"

Find result with the exact phrase, a proper name, or a set of words in a specific order

–term

Find result without term.

~term

Find result with term or one of its synonyms.

Currently supported on Web and Directory search.

Restrict Search

site:

Search only one website or domain.

link:

Find linked pages, i.e., show pages that point to the URL.

[#]..[#]

Search within a range of numbers.

Alternative Query Types

cache:

Display Google's cached version of a web page.

info:

id:

Find info about a page.

related:

List web pages that are similar or related to the URL.

Restrict Search to Sites where Query Words Appear

allinurl:

All query words must appear in the URL.

inurl:

The terms must appear in the URL of the page.

allinanchor:

All query words must appear in anchor text of links to the page.

inanchor:

Terms must appear in anchor text of links to the page.

allintext:

All query words must appear in the text of the page.

intext:

The terms must appear in the text of the page.

allintitle:

All query words must appear in the title of the page.

intitle:

The terms must appear in the title of the page.
